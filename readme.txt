Bento Blog Block is a custom Gutenberg block plugin for WordPress that provides a suite of blocks to construct “Bento-style” blog layouts, including image, video, post, social, map, text, container, gravatar, header, and link blocks with drag‑and‑drop support via React and React DnD 
It is designed to work with WordPress 6.4 or higher and requires at least PHP 7.2. The plugin’s stable version is 1.0.0, and it has been tested up to WordPress 6.7.1. The license is GPLv2 or later

== Features ==

**Post Block:** Displays a list or grid of blog posts with configurable layouts and pagination 
**Image Block:** Renders responsive images with optional lightbox support 
**Video Block:** Embeds self‑hosted or external videos (e.g., YouTube, Vimeo) with customizable controls 
**Link/Embed Block:** Creates styled hyperlinks or embeds arbitrary external content via URL 
**Map Block:** Integrates an interactive map (Leaflet-based) with markers and zoom controls 
**Social Block:** Displays social media icons/links with configurable styles 
**Text Block:** Offers rich text formatting consistent with other Gutenberg blocks 
**Container Block:** Allows nesting of multiple Bento blocks in grid or flex layouts 
**Gravatar Block:** Shows a user’s Gravatar avatar by email or user ID 
**Header Block:** Provides stylized page or section headers with background and overlay options

### Usage
Once activated, you can add any of the Bento blocks when editing a post or page in the Gutenberg editor. Each block has its own settings panel:

#### **Post Block:**
- Choose between list or grid layouts.
- Filter by category, tag, or custom taxonomy.
- Set pagination options (number of posts per page). 

#### **Image Block:**

- Upload or select an image from the Media Library.
- Configure alt text, caption, link URL, and lightbox toggle. 

#### **Video Block:**

- Paste a video URL (YouTube, Vimeo) or select a self‑hosted MP4/WebM file.
- Adjust autoplay, loop, and controls options. 

#### **Link Block:**

- Enter any valid URL or embed code.
- Choose from link styles (button, text link, card overlay). 

#### **Map Block:**

- Set center coordinates (latitude/longitude) and zoom level.
- Add multiple markers with popups that accept HTML. 

#### **Social Block:**

- Select from popular social networks (Facebook, Twitter, Instagram, etc.).
- Customize icon color, hover effect, and link target. 

#### **Text Block:**

- Standard rich text editing (headings, paragraphs, lists).
- Supports custom className and inline styles.

#### **Container Block:**

- Wrap multiple Bento blocks in a flex or grid container.
- Configure row/column gaps, alignment, and background. 

#### **Gravatar Block:**

- Input an email address or user ID to display the corresponding Gravatar.
- Set size (in pixels) and optional border style. 

#### **Header Block:**

- Enter a title and subtitle.
- Choose background image, overlay color, and text alignment.
